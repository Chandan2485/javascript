Take two numbers and only print the integer part when a is divided by b
Input Format
A two digit integer num is given as a function parameters.

Input Constraints
10 < a < 100
Output Format
An integer

Sample Input 1
32
Sample Output 1:
23
Explanation 1:
Reverse of 32 will be 23
Sample Input 2
94
Sample Output 2
49
Explanation 2
Reverse of 94 will be 49
